# Modding

Shadow Empire has limited modding capabilities added in with the release of the [S&S Editor](https://docs.google.com/document/d/1UCvS0p8mVXVwyjuaAfCTFxJ6p7VIbpPYfKDHzKcIHtY).

The goal of this site is to improve the available documentation for modders, and lower the barrier of entry for new modders.
