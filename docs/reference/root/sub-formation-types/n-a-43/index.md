# n/a

The 40M Nimrod was Hungarian anti-aircraft tank. Sporting a 40mm autocannon it was intended to be used as an anti-aircraft and anti-tank weapon, but it proved to be ineffective against Soviet T-34 tanks. Therefore, it was primarily utilized against lightly armored vehicles and for air defense. Its armor was also on the thin side (at best 28mm).
