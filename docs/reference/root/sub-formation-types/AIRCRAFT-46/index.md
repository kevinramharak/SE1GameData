# AIRCRAFT

The Messerschmitt Bf-109 was one of the first modern fighters employed in WW II and it formed the backbone of the German fighter force. Originally designed as a short-range interceptor it was also used as a bomber escort, fighter-bomber, ground-attack aircraft and reconnaissance plane.
