# Medium Tank

The Panzerkampfwagen IVf in this game models the PzIVd, PzIVe and PzIVf1. This is because all are similar in armor and all used the short-barelled 75mm gun. This was because this medium tank was originally intended to mainly take out fortified enemy infantry positions. Initially its main armaments were the short-barreled 75mm Kampfwagenkanone 37 L/24 optimized for high-explosive shells along with 2 or 3 7.92mm machineguns. The problem with the short-barrelled gun was it lacked the punch to be a threat to the T-34.
