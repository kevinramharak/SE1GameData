# Cavalry (it)

Cavalry is basicly mounted infantry. Contrary to some popular opinions cavalry in WW II did not charge into battle. Horses were used to quickly move to the battle and then dismounted from before engagements. Like infantry, cavalry performs best in defensive roles, especially when well entrenched.
