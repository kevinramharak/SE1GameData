# Wardog

Wardogs are descended from a breed of resurrected Dire Wolfs and are trained to be killers from a puppy onwards. Very dangerous to unarmoured infantry, and quite harmless to hard targets like tanks. Best used in an offensive role. Excellent recon value as well.
